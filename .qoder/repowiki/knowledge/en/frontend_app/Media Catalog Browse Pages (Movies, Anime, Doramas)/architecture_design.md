Each file is a self-contained React function component (`Movies`, `Anime`, `Doramas`) following an identical layout: local state holds `sort`, `genre`, `query` plus a debounced `debouncedQuery`; `@tanstack/react-query`'s `useInfiniteQuery` drives server-side pagination via `api.discoverMovies` / `discoverAnime` / `discoverDoramas`, with `getNextPageParam` advancing the page index; `react-intersection-observer`'s `useInView` triggers `fetchNextPage` when the sentinel element scrolls into view. All three pages render through the shared `SectionPageLayout` (title/subtitle/error/items/emptyMessage/footer) and `SectionToolbar` (search/sort/genre selectors), so each page only differs in its genre enum, sort options, query key prefix, and API call — no cross-page imports exist between them.
